
# Transcript Management System

A Java-based Transcript Management System that manages academic records,
calculates GPA, and sorts course grades using object-oriented design.

## Features
- Add and remove course grades
- Prevent duplicate grade entries
- GPA calculation based on credit hours
- Sorting using Comparable interface
- Clean and modular OOP design

## Technologies
- Java
- Collections Framework
- Comparable Interface
- Object-Oriented Programming
